# Introduction

基于 node+express+mongodb+vue 的注册登录前后端系统

## 功能

- [x] 前台登录注册/切换
- [x] Mongodb 数据库的使用
- [x] 用户注册逻辑
- [x] 基于数据库的注册验证和用户信息保存
- [x] 用户登录逻辑
- [ ] 使用 cookie 保存用户登录状态
